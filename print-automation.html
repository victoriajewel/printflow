Generate multiple products at once. Enter one product idea per line — PrintFlow will run each through the full Canva → Printify → Etsy pipeline automatically.

How batch mode works: Each line becomes one product. You can mix product types by prefixing lines with [tshirt], [mug], [hoodie]. Example:

[mug] Funny cat astronaut, "Space Catnip" text, pastel palette
